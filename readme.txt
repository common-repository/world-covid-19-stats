== Description ==

World Covid 19 stats will show corona current cases and more information about COVID-19. You will be able to show the relevant information: cases, deaths, recovered, and Active on you website.

No coding required. 

Put shortcode [wd_covid19] in backend page,post or widget.

shortcode support in sidebar using the ‘Custom HTML’ widget.

Compatibility with wordpress 5.4+

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can go through wordpress post/page backend.
